# Kanji Slasher MVP

漢字の力を操り、敵を薙ぎ払う2Dアクションゲーム。

## 起動方法
フォルダ内の `index.html` をブラウザ（Chrome推奨）で開いてください。
インストールやビルドは不要です。

## 操作方法
- **移動**: W / A / S / D
- **狙う**: マウスカーソル
- **通常攻撃（斬撃）**: 左クリック
- **能力発動**: 右クリック（アクティブ能力のみ）
- **ポーズ**: Esc キー

## ゲームの進め方
1. **漢字選択**: 初回プレイ時に「漢字」を選びます（変更不可）。
   - **火**: 火の弾を発射
   - **炎**: 周囲を焼き払う（常時）
   - **水**: 水の弾を発射
   - **氷**: 氷の弾を発射（敵を遅くする）
   - **土**: 土の弾を発射（貫通）
   - **金**: 右クリックで一定時間最強
   - **鉄**: 常時攻撃力アップ（堅実）
   - **毒**: 周囲を毒にする（常時）
2. **ステージ**: 全3ステージ。各ステージの最後にボスが出現します。
3. **強化**: 敵を倒してMoneyを集め、ステージクリア後に能力を強化できます。
